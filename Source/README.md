# Datasheet Evidence Source

This folder contains only local manufacturer datasheet PDFs that are cited by component pinout documentation or needed as local source evidence for future wiring work.

Do not keep temporary AllDatasheet duplicates, failed downloads, or `Zone.Identifier` sidecar files here. If a pinout doc cites a local PDF, keep exactly that evidence file.
