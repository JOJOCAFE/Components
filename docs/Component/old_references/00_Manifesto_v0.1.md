# Components Manifesto v0.1

## Vision

Components is an open platform for understanding electronic systems through reading, building, and simulation.

## Core Principles

Human First.
Read Before Learning.
Easy to Begin.
Unlimited Possibilities.
Component Describes.
Board Presents.
Device Executes.
Topology, Then Validation.

## Philosophy

Easy to read.
Understanding before syntax.
Learning before memorization.
Complexity only when needed.
