# MIL gate PNGs with pins

Raster counterparts of `../mil-with-pins/`, imported from
[`cludewn/logic-gate-symbols`](https://github.com/cludewn/logic-gate-symbols)
at `mil/png/with-pins` on 2026-07-15. Upstream declares CC0-1.0; this import
is pinned to `72a6e9caa3d50ba1d6de8ec08051c36b87410c2d`.

These PNGs are fallback or thumbnail artwork only. Prefer the paired SVG for
scalable Board/editor rendering. Neither form defines Component/Device ports,
logic, timing, or connectivity.
