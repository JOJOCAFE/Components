# MIL gate symbols without pins

These SVG files are imported presentation assets from
[`cludewn/logic-gate-symbols`](https://github.com/cludewn/logic-gate-symbols),
`main/mil/svg/no-pins`, retrieved on 2026-07-15. The upstream repository
declares the set **CC0-1.0**. This import is pinned to upstream commit
`72a6e9caa3d50ba1d6de8ec08051c36b87410c2d`.

Included symbols: AND, bubble, buffer, NAND, NOR, NOT, input-negative NOT,
OR, XOR, and XNOR. Use these when an editor supplies its own port terminals
and labels; use `../mil-with-pins/` when the picture should include generic
lead stubs.

These files are artwork only. A gate editor must source its ports, directions,
logic behavior, timing, and connectivity from the validated resolved
Component/Device contract, never from SVG geometry.
